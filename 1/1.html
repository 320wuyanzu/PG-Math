# A、函数

考试内容：

- 函数的概念及表示法
- 函数的有界性、单调性、周期性和奇偶性
- 复合函数、反函数、分段函数和隐函数
- 基本初等函数的性质及其图形
- 初等函数
- 函数关系的建立

考试要求：

- 理解函数的概念，掌握函数的表示法，并会建立应用问题的函数关系．
- 了解函数的有界性、单调性、周期性和奇偶性．
- 理解复合函数及分段函数的概念，了解反函数及隐函数的概念．
- 掌握基本初等函数的性质及其图形，了解初等函数的概念．

函数是一种映射，从定义域 DfD_fDf​（⊂R\subset R⊂R） 到值域 RRR（实数集）的映射

两个函数相同 ⟺\Longleftrightarrow⟺ 它们的定义域、对应法则都相同

# 1、有界性

总能找到某个正数 KKK，使得 −K⩽f(x)⩽K-K \leqslant f(x) \leqslant K−K⩽f(x)⩽K

即函数值始终不会超过某个区间

有界性实际上表达的是函数值的范围，在图形上表现为函数曲线能被两条直线夹在中间

换种说法：总能找到两个数 K1<K2K_1 < K_2K1​<K2​，使得 K1⩽f(x)⩽K2K_1 \leqslant f(x) \leqslant K_2K1​⩽f(x)⩽K2​

# 有上界

总能找到某个数 K1K_1K1​，使得 f(x)⩽K1f(x) \leqslant K_1f(x)⩽K1​

即函数值始终小于 K1K_1K1​

# 有下界

总能找到某个数 K2K_2K2​，使得 K2⩽f(x)K_2 \leqslant f(x)K2​⩽f(x)

即函数值始终大于 K2K_2K2​

# 如何判断有界性？

  • 函数有界 ⟺\Longleftrightarrow⟺ 既有上界，也有下界
  • 连续函数的最值定理：若函数在某个闭区间上连续，则它在该区间上有界，且一定能取得它的两个最值

讨论有界性时，如果未指明区间，那么一般默认是在定义域上讨论

# 2、单调性

  • 单调增加：如果任意 x1<x2x_1 < x_2x1​<x2​，有 f(x1)<f(x2)f(x_1) < f(x_2)f(x1​)<f(x2​)
  • 单调减少：如果任意 x1<x2x_1 < x_2x1​<x2​，有 f(x1)>f(x2)f(x_1) > f(x_2)f(x1​)>f(x2​)

单调性实际上表达的是函数值的变化趋势，单增说明函数值一直在变大，单减说明函数值一直在减小

讨论单调性时要明确讨论区间

# 如何判断单调性？

  • 利用定义判断
  • 利用一阶导数判断

# 3、奇偶性

函数的定义域必须关于原点对称，否则讨论个锤子

  • 奇函数：f(x)=−f(−x)f(x) = -f(-x)f(x)=−f(−x)，明显函数图像关于原点对称
  • 偶函数：f(x)=f(−x)f(x) = f(-x)f(x)=f(−x)，明显函数图像关于 yyy 轴对称

# 如何判断奇偶性？

利用定义判断

# 4、周期性

若存在正数 TTT，使得任意的 f(x)=f(x+T)f(x) = f(x+T)f(x)=f(x+T)，那么 TTT 就是函数的一个周期

一般说的周期，指的是最小正周期，但是有的周期函数没有最小正周期

# 5、反函数、复合函数

反函数是逆映射的特例，
它和原函数的单调性相同，
它与原函数的图像关于直线 y=xy=xy=x 对称

函数复合时，要注意顺序，f[g(x)]f[g(x)]f[g(x)] 有意义，不代表 g[f(x)]g[f(x)]g[f(x)] 有意义
